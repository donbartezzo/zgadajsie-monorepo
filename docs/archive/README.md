# Archiwum dokumentacji

Ten katalog zawiera dokumenty historyczne, przestarzałe lub jednorazowe.

## Zasady użycia

- pliki z tego katalogu **nie są aktywnym source of truth** dla AI
- przed odwołaniem się do dokumentu z archiwum należy najpierw sprawdzić aktualny kod i aktywną dokumentację z `docs/`
- jeśli wiedza z archiwum nadal jest potrzebna operacyjnie, należy ją przepisać do aktywnego dokumentu zamiast używać pliku historycznego bezpośrednio
